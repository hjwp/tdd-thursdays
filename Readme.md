# Setup


* assumes you're running node v10.

If you're using nvm:

```sh
nvm use .
```

## install jasmine:

```sh
npm install
```


## run the tests:

```sh
make test
# or
npm test
```

## auto-rerun:

```sh
make watch-tests
```

## find our files:

* tests live in `spec/Specs.js`
* source lives in `lib/index.js`
